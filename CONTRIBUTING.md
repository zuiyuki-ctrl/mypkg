# Contributing

Thank you for your interest in contributing to System Monitor!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Ensure all tests pass: `colcon test`
5. Submit a pull request

## Code Style

Please follow PEP 8 style guidelines for Python code.

## Testing

Before submitting a pull request, please ensure:
- All linter tests pass (flake8, pep257)
- Copyright headers are present in all source files
- The package builds successfully
